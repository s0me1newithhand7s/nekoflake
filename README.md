# nekoflake

simple flake for `nekoray` (3.x) and `nekobox` (4.x) with [garnix](https://garnix.io) cache. 
[original](https://github.com/MatsuriDayo/nekoray) repository got archived, and `nekobox` was not merged into `nixpkgs`.
